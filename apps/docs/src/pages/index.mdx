# Introduction

Envless is the end-to-end encrypted, open source and frictionless way to share and manage app secrets (**env variables**) across teams, platforms, infrastructures, CI/CD. It's language and framework agonistic and fits right into your development and deployment workflows. {{ className: 'lead' }}

## Getting started {{ anchor: true }}

To get started with Envless, you need to [create an account](https://envless.dev/signup) and create a project. {{ className: 'lead' }}
