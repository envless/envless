## ENV Variables
Envless CLI allows you to manage your project's ENV variables with the following commands:

| Command | Description |
|:--------|:--------|
| `envless env get KEY1 KEY2` | Get the values of the specified ENV variables. |
| `envless env create KEY=value` | Create a new ENV variable with the specified key and value. |
| `envless env update KEY=value2` | Update the value of an existing ENV variable with the specified key. |
| `envless env delete KEY1 KEY2` | Delete the specified ENV variables. |
| `envless env rename KEY1 KEY2` | Rename an existing ENV variable from KEY1 to KEY2. |
